# Kod-04

Split Wireless keyboard with trackpoint (SK8707-01)

Based on Auto-KDK design

## Features

- 80 keys (40 per side)
- SK8707-01 Trackpoint on right side
- Rotary encoder on left side
- Cursor keys on left side
- Mouse buttons on right side
- ZMK firmware
- XIAO nRF52840 Plus controllers
- 200mAh battery (402030)

## Parts List

| Part | Quantity |
| --- | --- |
| XIAO nRF52840 Plus | 2 |
| SK8707-01 Trackpoint | 1 |
| Rotary Encoder (EC11) | 1 |
| Battery 402030 (200mAh) | 2 |
